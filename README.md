# About 

This is webpage for the tech company called Netnova tech company in this project i will be using claude for UI node js for backend github action for CI/CD and aws for deployment 

# Problem
  1. Client wanted a landing webpage to show their services
  2. Client like to host their website in cloud
  3. Client want a service where they can make changes and deploy as fast as possible 

# solution 

  1. The UI design of landing page is created with the help of claude ai which showcase the prompt engineering
  
  2. To solve the problem of CI/CD i used github action to create a workflow
  
  3. To deploy webpage i used AWS platform using ec2 instances where i use general purpose instance as client is new startup and they only wanted to check the traffic flow to their webpage but not for long term commitment  after a while i will use more cost effective instances if traffic flow is of good amount.
